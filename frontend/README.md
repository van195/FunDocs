## Frontend (React + Vite)

### Setup
1. `cd frontend`
2. `npm install`
3. `copy .env.example .env` (edit only if needed)
4. `npm run dev`

### Features
- Register/Login (JWT stored in `localStorage`)
- Chapa paywall gate
- Upload `pdf/txt/docx`
- Chat about the active uploaded document
- “Fun Mode” toggle
